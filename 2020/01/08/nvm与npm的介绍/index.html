一、NVM

 nvm 是 node version manage 的简写，直译的意思就是 node 的版本管理工具。

1、安装步骤

1.1、下载NVM for Windows

https://github.com/coreybutler/nvm-windows/releases

这里选择 nvm-setup.zip 这个选项

nvm-setup.zip：这是一个安装包，下载之后点击安装，无需配置就可以使用。


(1)、 解压后运行安装包，其中有一步是 ： Set Node.js Symlink

这里是设置 Node.js 的快捷方式存放的目录，该目录是一个快捷方式，指向通过 nvm 下载的 Node.js 存放的位置，也就是 nvm 目录里的文件,也就是下图框红的部分。

(2)、安装完成后在cmd界面输入 nvm ，如果出现 nvm 版本号和一系列帮助指令，则说明安装成功

二、NPM

 npm 是 node package manage 的简写，直译的意思就是 node 的包管理工具。

注意：
在每个版本的nodejs中，都会自带npm，为了统一起见，我们安装一个全局的npm工具，这个操作很有必要，因为我们需要安装一些全局的其他包，不会因为切换node版本造成原来下载过的包不可用。

1、安装步骤

(1)、进入 cmd 界面，输入：

这个全局安装路径是使用 -g 安装包时的地址，一般设在安装 nvm 目录下的 npm 中。


(2)、执行完成后会在 C盘 / 用户目录 / 用户名 下生成一个 .npmrc 的文件

打开后输入:

prefix：是 Node.js 包的全局安装路径
cache：是 Node.js 包的缓存路径


(3)、在 cmd 命令行中输入


(4)、为这个 npm 配置环境变量

变量名： NPM_HOME
变量值： npm 的全局安装地址


(5)、在 Path 的最前面添加 %NPM_HOME% 这个一定要添加在 %NVM_SYMLINK% 之前，所以直接把 %NPM_HOME% 放到 Path 的最前面


(6)、安装 cnpm

因为天朝墙的原因，有时使用 npm 下载会很慢，这时候使用 国内的镜像 cnpm 下载会快一些
