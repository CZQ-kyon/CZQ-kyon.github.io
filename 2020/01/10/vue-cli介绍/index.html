 使用 Vue-cli 构建出来的是单页web应用的项目，整个项目中只有一个 index.html 文件，其他网页都是通过 组件中的模板 在 App.vue 上的 <router-view /> 渲染出来的。

在 vue-cli 搭建的项目中，有几个比较重要的文件，分别是：

  • main.js ：入口文件

  • App.vue ：主组件

  • router / index.js ：路由文件

  • components 文件夹 ：组件文件夹，用来存放要在 App.vue 中的 <router-view /> 展示的模板（.vue）。

一、安装vue-cli

使用 npm 或 cnpm 全局安装 vue-cli

安装完后，使用 -V 检查 vue-cli 版本，如果出现版本号，则说明安装成功了。

使用 vue init 来初始化项目：

<template-name> : 模板名称，有5种模板：

  • webpack ：一个全面的webpack+vue-loader的模板，功能包括热加载，linting,检测和CSS扩展。

  • webpack-simple ：一个简单webpack+vue-loader的模板，不包含其他功能，让你快速的搭建vue的开发环境。

  • browserify ：一个全面的Browserify+vueify 的模板，功能包括热加载，linting,单元检测。

  • browserify-simple ：一个简单Browserify+vueify的模板，不包含其他功能，让你快速的搭建vue的开发环境。

  • simple ：一个最简单的单页应用模板。

  1. cd test 进入 vue 的项目目录。

  2. cnpm install 安装项目依赖包，也就是 package.json 里的包。

  3. npm run dev 开发模式下运行程序。自动构建了开发用的服务器环境，并实时监视代码的修改，并呈现在网页中。

三、vue-cli 项目结构

1、package.json

存放项目开发所需的各种模块。

执行后会在项目根目录生成 dist 文件夹 ， 将这个文件传到云服务器上就可以实现云端访问。


1.2、dependencies字段和devDependencies字段

  • dependencies : 项目运行时所依赖的模块，发布后还需要用到的模块 （npm install xxx –save）

  • devDependencies : 项目开发时所依赖的模块 （npm install xxx –save-dev）


2、.editorconfig

定义项目的编码规范，编译器的行为会与.editorconfig文件中定义的一致，它的优先级比编辑器自身设置的高。


3、index.html

整个项目中只有这个 .html 文件

index.html 文件中用了一个 id 为 app 的 div 标签包裹了一个 <router-view/>标签。


4、src / main.js 入口文件

main.js是整个项目的入口文件

这个 main.js 中的 el:'#app' ,说明控制的是 id 为 app 的标签包裹的位置，也就是 index.html 中的 <router-view/> 。 通过 import App from './App' 引入根目录下的 App.vue 中 <template> 中写的 html 代码 , 替换 <div id="app"></div> 里的内容。


5、App.vue 组件

此处就是初始完项目后 ， npm run dev 在页面中看到的初始 vue 的项目的内容，其中的 <router-view></router-view> 展示的就是 router / index.js 路由文件下路径为 path ： ‘/‘ 时，组件的内容 component : Hello ，Hello.vue 中 <template> 中写的 html 代码。


6、router / index.js 路由文件

路由文件就是 通过监听 url 然后选择展示 component 文件夹里的 .vue 文件

当有多个 .vue 文件时，就在这个路由文件的 routers 里写上：在路径为 xxx 时，展示什么组件。

需要注意的是：在使用 component 前要先将 .vue 引入 import xxx from '.vue文件的位置'，不然会使用失败。

所有的 .vue 文件都是显示在 App.vue 下的那个 <router-view> 中

例：

当 url 为 / 时，展示 Login.vue；
当 url 为 /administratorWeb 时，展示 administratorWeb.vue；
当 url 为 /parentWeb 时，展示 parentWeb.vue。
