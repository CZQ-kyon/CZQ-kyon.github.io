十二、用 Proxy 进行预处理

1、声明Proxy

第一个 { } 相当于方法的主体，后面的 { } 是 Proxy 代理处理区域，相当于写钩子函数的地方


2、get属性

get属性是在的得到某对象属性值之前进行的预处理的方法，接收三个参数

  • target：得到的目标值，大括号中的内容

  • key：目标的Key值，相当于对象的属性

  • property


3、set属性

set属性是要改变 Proxy 属性值时，进行的预先处理，接收四个参数

  • target：目标值

  • key：目标的Key值

  • value：要改变的值

  • receiver：改变前的原始值


4、apply使用

作用是调用内部的方法，使用在 方法体是一个匿名函数时
